Doily Generator

Welcome to the Doily Generator! A tool that creates simple doily patterns. The patterns are all in crochet diagram format with an included legend. Feel free to use the generated pattern as is, or modify it as you see fit.

The doily patterns are not saved anywhere and cannot be retrieved, so I recommend taking a screenshot of the ones you like.

Note: Each row will always start with a chain 3 and end with a slip stitch into the third chain. The chain 3 is highlighted in red to make it easy to see in the diagram.

Disclaimer: All the patterns should be workable without any modifications needed. If you encounter some that contain mistakes such as a missing stitch or stitch that works into nothing, please log it here with a picture. I recommend looking over the whole pattern before starting and if you see anything out of place, you might be able to make an easy fix (e.g. by adding a missing stitch or replacing a dc with a chain) or generate a new one.






Setting Description
Row Count Slide to set the number of rows that the doily will contain (between 5 and 15 rows).
Section Count Select the number of repeating sections for the doily. For example, a section count of 4 means that the same pattern will repeat 4 times for each row.
Show Chart When checked, shows the crochet diagram that can be used to create the pattern. When unchecked, shows an approximation of what the finished product will look like.
Show Left Handed When checked, shows the pattern to be worked for left handed users (pattern goes clockwise). When unchecked, shows the pattern for right handed users (pattern goes counter clockwise).
Generate Click to generate a new doily using the above settings.
Random Click to generate a new doily with the above settings randomized.
